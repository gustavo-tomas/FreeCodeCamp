Introduction

The c++ (also reffered as cpp) programming language is an imperative, object oriented, generic language widely used in many applications due to its efficiency. In this page, some key aspects of the syntax will be explained.

Applications

As c++ is a compiled language, as opposed to an interpreted one such as python, it is very efficient. Moreover, it doesn't feature some high level concepts and abstractions (such as inferring a variable type) gaining some complexity in exchange for speed. That being said, features such as object oriented paradigm grants a mix between efficiency and versatility desired by many applications. Therefore, applications that require fast executions greatly benefit from c++'s design. Below is a list of common uses of cpp:

  • Video Games
  • Physics Simulations
  • Operating Systems
  • Physics Engines
  • Mobile applications
Variables and Constants

One of the key aspects of any programming languages, variables and constants are essencial for making calculations and storing values. Variables are terms that can store a specific value, such as float, boolean, integer or char. Here is how to declare a variable of type integer named "this is an integer" with value 0:

int this_is_an_integer = 0;

A constant is similar in declaration to a variable, but once the value of the constant has been set, it cannot be changed. Here is how to declare the same example as before:

const int this_is_an_integer = 0;
Conditional Statements

Conditional statements are statements made to execute a given code if a certain condition is met. For example, the code below adds 1 to this_is_an_integer if it is equal to zero (here equality is expressed by "=="):

if(this_is_an_integer == 0) { this_is_an_integer += 1; }
Loop Statements

Loop statements' behaviour is to repeat a certain action until a condition is met. There are two main types of loops in c++: for and while. Both do the same job but in different ways.

The for loop syntax consists of three elements: a starting condition, an ending condition and an action to do at the end of each loop. The code below adds 1 to our variable as long as it is less than 10:

for(this_is_an_integer = 0; this_is_an_integer < 10; this_is_an_integer++) { /* more stuff to do */ }

The while loop only contains a condition. The loop than repeats until the condition is not met. The code below works just like before:

while(this_is_an_integer < 10) { this_is_an_integer++; }
